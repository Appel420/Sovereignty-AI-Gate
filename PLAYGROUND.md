# PLAYGROUND MODE

This repository is a playground for iterative experimentation, local prototyping, and rapid validation.

## Rules
- Changes should remain reviewable and easy to revert.
- Local-only behavior is preferred by default.
- No hidden synchronization or silent external persistence.
- User consent is required before any shared memory leaves its scoped boundary.
- Blocked providers or models must not appear in selection or collaboration flows.

## Intent
This repository is intended for collaboration, testing, and refinement. If a direction proves unhelpful, it can be discarded and replaced without affecting unrelated work.
