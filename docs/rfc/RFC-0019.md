# RFC-0019 — Import Bundle

**Status:** Active  
**Version:** 0.1.0  

## Abstract

Defines the import bundle validation and loading process.

## Specification

Import bundles are structurally identical to export bundles with the addition of
`imported_at`. Before loading:

1. Schema validation (required fields present, version supported).
2. `payload_hash` integrity check.
3. Duplicate detection (`bundle_id` uniqueness).

Signature verification (step 4) requires the signer's public key and is out of scope
for this RFC; it is handled by the caller.

## Conformance

- Unsupported schema version MUST raise `E_IMPORT_VERSION_UNSUPPORTED`
- Duplicate bundle_id MUST raise `E_IMPORT_DUPLICATE`
- `payload_hash` mismatch MUST raise `E_IMPORT_SCHEMA_INVALID`
