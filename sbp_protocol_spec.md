# SBP Protocol Specification Pointer

This file points to the normative core protocol specification:

- `docs/rfc/SBP-0000-Core-Principles-and-Terminology.md`

Implementations SHOULD treat SBP-0000 as the foundational conformance contract.

## Normative scope update

SBP-0000 includes core normative requirements for provider independence,
explicit egress governance, local authority, portable data, and transparent
external dependencies (SBP-A11 through SBP-A15).
